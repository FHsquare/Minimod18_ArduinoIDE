# Minimod18_ArduinoIDE
Use minimod18 with Arduino IDE

minimod18 was project by Elektor Magazine in associatioon with CC2.
It was presented in Issue 1 of 2010.

For ccompiling and uploading use following settings:

Board: Arduino Nano
Processor: Atemga328p
Programmer: USBAsp

You need to use upload by programmer, since the minimod18 has no serial converter.
